# Raw Data

This folder documents the raw datasets used by the data pipeline.

## Files

| File | Description | Rows | Source |
|------|-------------|------|--------|
| train.csv | Layer 1 training set (CEX 2022) | 59,800 | Generated by ingestion pipeline from BLS CEX PUMD |
| production.csv | Data generator seed (CEX 2023) | 61,428 | Generated by ingestion pipeline from BLS CEX PUMD |
| evaluation.csv | Evaluation set (CEX 2024) | 62,900 | Generated by ingestion pipeline from BLS CEX PUMD |
| evaluation_moneydata.csv | Long-running eval set (MoneyData) | 6,079 | MoneyData dataset |

## Location

All files are stored in MinIO at `s3://data/raw/` on the Kubernetes cluster.

## Reproducing

Run the ingestion pipeline to regenerate these files:
```bash
cd data_pipeline/ingestion
python3 generate_transactions.py --year 2022 --output train.csv
python3 generate_transactions.py --year 2023 --output production.csv
python3 generate_transactions.py --year 2024 --output evaluation.csv
```
